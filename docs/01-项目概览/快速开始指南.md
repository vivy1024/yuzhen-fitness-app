## 🎯 快速启动概述

本指南帮助您快速启动build_body智能健身助手系统，包括AutoGen+MCP完整架构的配置和运行。

### ⚡ 一键启动方式

```bash
# 克隆项目
git clone <repository-url>
cd build_body

# 一键启动完整系统
python start_autogen_mcp_system.py
```

**启动后访问地址**:
- AutoGen协调员: http://localhost:5000
- 前端应用: http://localhost:9000
- API服务: http://localhost:8000
